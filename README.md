# rust-solid-app

### Apps and Packages

-   `api` rest web server.
-   `web` web client application.

### Requirements

-   [api](./apps/api/README.md)
-   [web](./apps/web/README.md)

### Usage

#### Run `api`

```bash
pnpm api:dev
```

#### Run web app

```bash
pnpm dev
```

### Build

```bash
pnpm build
```
